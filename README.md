## 命名規則
単語をハイフン（-）で区切る形式（ケバブケース）を採用します。　例：kebab-case
命名規則とは、クラスや変数などの名前を自分で決める時のルールです。みんなに伝わるわかりやすい名付けを心がけましょう。
